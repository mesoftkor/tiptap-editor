## [1.0.1] - 2025-11-06

### Fixed
- 다크모드에서 에디터 본문 텍스트가 검은색으로 표시되어 가독성이 떨어지는 문제 수정
- 다크모드에서 모든 텍스트 요소(제목, 문단, 리스트)의 색상을 밝은 색상으로 변경
- 다크모드에서 링크 색상 개선
- 다크모드에서 하이라이트(형광펜) 색상 개선

### Added
- 버전 관리 시스템 (Semantic Versioning)
- CHANGELOG.md 파일 추가
- 릴리즈 자동화 스크립트
- VERSIONING.md 가이드 문서

### Changed
- package.json에 repository 정보 추가
- README.md에 GitHub 설치 방법 추가
- 빌드된 dist 파일을 Git에 포함하여 GitHub 설치 지원

## [1.0.0] - 2025-11-03

### Added
- 초기 릴리즈
- React, Next.js, Svelte 지원
- S3 presigned URL 업로드 기능
- 슬래시 명령어 메뉴
- 다크 모드 지원
- 이미지 업로드 및 삭제 기능
- 사용하지 않는 이미지 추적 기능
- TypeScript 타입 정의
- 예제 프로젝트 (React, Next.js, Svelte)
